## Reactions
---
A small collection of animations for small scale projects. 

### Usage
You are able to import directly from the main package:
```import { fadeIn } from 'reactions'```


### Animations
---
List of Available Animations:
- Fade
- Slide 
- Bounce
- Flash 
- Rotate

Animation Qualities:
- In
- Out
- InPlace
- Right
- Left
- Up
- Down
- Trigger
